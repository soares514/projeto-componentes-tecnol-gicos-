# projeto-componentes-tecnol-gicos-Sistema Inteligente de Monitoramento de Salas de Aula

Descrição

Este projeto apresenta um sistema de monitoramento de salas de aula utilizando componentes tecnológicos e conceitos de Internet das Coisas (IoT). O objetivo é coletar dados do ambiente e disponibilizá-los em tempo real para análise.

Componentes Tecnológicos

Hardware

- Sensor de temperatura
- Sensor de luminosidade
- Sensor de presença
- ESP32 ou Arduino

Software

- Arduino IDE
- Banco de dados
- Dashboard Web

Comunicação

- Rede Wi-Fi
- Protocolo MQTT ou HTTP

Funcionamento

Os sensores coletam informações do ambiente e enviam os dados para o microcontrolador. Em seguida, as informações são transmitidas para um banco de dados e exibidas em uma interface web para acompanhamento em tempo real.

Benefícios

- Monitoramento ambiental.
- Economia de energia.
- Melhoria do conforto dos usuários.
- Aplicação prática de IoT.
